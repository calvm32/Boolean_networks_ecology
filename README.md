# Boolean_networks_ecology

## Run instructions

To make and activate the virtual environment, run
```
python3 -m venv .venv
source .venv/bin/activate
```

To get everything you need, run 
```
pip install -e .
pip install -r requirements.txt
```

To actually start working, run
```
export PYTHONPATH=C:\Users\Me\Downloads\Boolean_networks_ecology:$PYTHONPATH
